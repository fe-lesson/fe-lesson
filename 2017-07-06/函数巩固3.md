# 函数巩固3

## 题目

请写一个函数，函数有一个参数 n（n为正整数），数列 a0 = 1; a1 = 2; a2 = a0 + a1 = 3; a3 = a2 + a1 = 5;.... 请输出 an 的值;


#### One
```js
// 递归
function fibonacci(n){
  if (n==1){ 
    return 1;
  }
  else if (n==2) {
    return 2;
  }
  else {
    return fibonacci(n-2) + fibonacci(n-1);
  }
}

fibonacci(5);
```

function fibonacci(n) {
  const arr = [1, 2];

  if (n <= 1) {
    return arr[n];
  }

  for (let index = 2; index <= n; index++) {
    const newNum = arr[index - 1] + arr[index - 2];

    arr.push(newNum);
  }

  return arr[n];
}

fibonacci(5);
